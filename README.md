# Music Library

Повністю робочий проект для завантаження, прослуховування та скачування музики.

## Функціонал
- Реєстрація та авторизація користувачів (JWT)
- Завантаження аудіофайлів
- Список треків з можливістю прослуховування та завантаження
- Видалення треків (тільки для власника або адміністратора)
- База даних SQLite

## Важливі поради
- Переконайтеся, що користувачі завантажують тільки музику, на яку мають права.
- Для продакшену замініть JWT_SECRET на довгий випадковий рядок.
- Розгляньте використання S3 або іншого хмарного сховища для аудіо-файлів.
- Регулярно робіть бекапи бази `data.sqlite3`.
